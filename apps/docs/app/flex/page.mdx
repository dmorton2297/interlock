# Flex
*Interlock flex component* <br />
This component inherits all params of the Box component<br />
🔗 [justify](#justify) ▪️ [direction](#direction)

<div id="inherited">
  #### Inherited props
  Box 🔗 [children](/box#children) ▪️ [padding](/box#padding) ▪️ [margin](/box#margin) ▪️ [css](/box/#css) ▪️ [className](/box/#css)<br />
</div>

## Props
```typescript
export interface FlexProps extends BoxProps {
  /**
   * @param children children that are passed into the box
   */
  justify?: "between" | "around" | "center" | "end" | "start";
  /**
   * @param direction direction the flexbox should operate in
   */
  direction?: "row" | "column";
}
```
